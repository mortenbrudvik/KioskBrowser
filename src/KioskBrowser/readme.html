Swift Kiosk Browser

Swift Kiosk Browser is a lightweight, single-instance web browser designed for seamless integration with command-line workflows. Leveraging Microsoft WebView2, it offers a fast and secure browsing experience. Easily launch the browser from the command line to display any specified web page.

Usage

KioskBrowser.exe "http://www.google.com"

Options:

Option Description
-t, --enable-titlebar Enable Title bar
-r, --enable-content-refresh (default: disabled) Enable automatic refresh of content
--content-refresh-interval (min: 10, max: 3600, default: 60) Content refresh interval in seconds

Close window by pressing ESC. (Disabled when title bar is enabled)
